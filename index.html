Comprehensive Protection

Every feature is engineered to provide maximum security while respecting your privacy and context.

Security & Authentication

  • Google Sign-In: Seamless & secure account access.
  • Biometric & PIN: App Lock with Fingerprint/Face ID.
  • Emergency Codes: Tri-state PINs (Green: Cancel, Yellow: Alert, Red: SOS) for duress situations.

Smart Safety Engine

  • 4-Stage Escalation: Auto-progresses: Check-in → GPS → Call Contacts → Dial 112.
  • Shake to SOS: Instantly trigger emergency mode.
  • Fake Call: Simulate incoming calls to safely exit uncomfortable situations.

Location & Navigation

  • Live Sharing: Real-time 30s interval updates to trusted contacts.
  • Route Monitoring: Safe Walk mode alerts upon path deviation or entering Risk Zones.
  • Reverse Timer: "I'll arrive in X minutes" automated safety countdowns.

Anti-Theft Protection

  • Intruder Capture: Auto-snaps front camera photo on wrong PIN.
  • Hardware Monitoring: SIM change detection & stealth mode execution.
  • Theft Deterrents: Fake blank screen, warning overlays, and loud alarms.

Cloud Sync & Backup

  • Firebase Auto-Sync: Debounced & throttled backups of all your settings.
  • Cross-Device Restore: Seamless data restoration when logging into a new device.
  • Granular Controls: Choose exactly what to sync (contacts, zones, settings).

Evidence & Social

  • Encrypted Vault: Secure local storage of safety events & GPS trails.
  • Report Generation: Export timestamped evidence as structured PDFs.
  • Family Mapping: View family locations & organize emergency contact groups.

Privacy by Design

No hidden surveillance. We prioritize your consent for all tracking features.

Encrypted Local Storage. Your evidence vault is securely locked on your device.

Opt-In Sharing. All location sharing is entirely user-controlled and transparent.

Granular Sync. Cloud data is only synced when signed in, giving you control over what leaves your device.

Robust Architecture

  • Offline-First Capabilities

    Core safety functionalities execute seamlessly without internet using Hive local storage.

  • Provider Pattern

    Clean, scalable state management ensuring snappy, reactive UI updates across the app.

  • Native Platform Channels

    Custom Kotlin (Android) & Swift (iOS) implementations for deep hardware integration like SIM monitoring.

Developer Setup

Prerequisites: Flutter SDK ≥ 3.x, Dart ≥ 3.x, and Firebase configurations.

bash
# Clone the repository
git clone https://github.com/g1rao/SilentGuardian.git
cd SilentGuardian

# Install dependencies
flutter pub get

# Run on Android
flutter run

# Run on iOS
cd ios && pod install && cd ..
flutter run -d ios

Firebase Integration Steps

  1. Create a Firebase project at console.firebase.google.com
  2. Add Android app with package: com.silentguardian.silent_guardian
  3. Add iOS app with bundle ID: com.silentguardian.silentGuardian
  4. Download configs: google-services.json → android/app/ | GoogleService-Info.plist → ios/Runner/
  5. Enable Google Sign-In in Authentication → Sign-in method
  6. Create Firestore Database in test mode and enable Firebase Storage
